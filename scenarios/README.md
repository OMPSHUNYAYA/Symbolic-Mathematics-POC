# Scenarios

This folder contains the ten minimal Python scripts used by the Getting Started pages.

- `scenario_01_two_sensors.py`
- `scenario_02_kpi_rollup.py`
- `scenario_03_three_sensors.py`
- `scenario_04_regression_outlier.py`
- `scenario_05_imaging_focus_burst.py`
- `scenario_06_activation_energy.py`
- `scenario_07_mac_chain.py`
- `scenario_08_forecast.py`
- `scenario_09_climate_faulty_sensor.py`
- `scenario_10_robotics_torque.py`

> Note: `scripts/run_all.py` stays in the `scripts` folder. Individual scenario files live here in `scenarios/`.
